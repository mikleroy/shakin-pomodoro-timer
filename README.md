# 🍅 Pomodoro Timer — Расширение для Chrome

Таймер по технике помидоро с историей в виде тепловой карты и синхронизацией через GitHub Gist.

## Установка

1. Скачай и **распакуй архив** `pomodoro-extension.zip` в любую папку на компьютере
2. Открой Chrome и перейди на страницу: `chrome://extensions/`
3. Включи **«Режим разработчика»** (переключатель в правом верхнем углу)
4. Нажми **«Загрузить распакованное расширение»**
5. Выбери папку `pomodoro-extension`
6. Иконка 🍅 появится в панели расширений

## Использование

- **Кнопка «старт»** — запускает 25-минутный рабочий цикл
- По завершении цикла появится уведомление, автоматически запустится 5-минутный перерыв
- В иконке расширения отображается обратный отсчёт в минутах (красный = работа, зелёный = перерыв)
- Вкладка **«История»** — тепловая карта по месяцам за выбранный год

## Подключение GitHub Gist (синхронизация в облако)

1. Зайди на [github.com/settings/tokens/new](https://github.com/settings/tokens/new)
2. В поле **Note** введи любое название (например, `Pomodoro Timer`)
3. Поставь галочку **`gist`** в списке прав
4. Нажми **Generate token** и скопируй токен (он начинается с `ghp_`)
5. В расширении перейди в **Настройки**, вставь токен и нажми **Сохранить**
6. История автоматически создаст приватный Gist и будет синхронизироваться после каждого помидора

## Цветовая шкала (тепловая карта)

| Цвет | Помидоров в день |
|------|-----------------|
| ⬛ тёмный | 0 |
| 🟫 коричневый | 1 |
| 🟧 тёмно-оранжевый | 2–3 |
| 🟥 оранжево-красный | 4–5 |
| 🔴 красный | 6–8 |
| 🔴 яркий красный | 9+ |

## Экспорт и импорт

Во вкладке **Настройки** можно:
- Экспортировать всю историю в JSON-файл (резервная копия)
- Импортировать историю из JSON-файла
