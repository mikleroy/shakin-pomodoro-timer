# 🍅 Shakin Pomodoro Timer — Chrome Extension

Таймер по технике помидоро с историей в виде тепловой карты.

## Установка

### Из Chrome Web Store
*(ссылка появится после публикации)*

### Для разработки (unpacked)

1. Склонируй репозиторий или скачай ZIP
2. Открой Chrome → `chrome://extensions/`
3. Включи **«Режим разработчика»** (правый верхний угол)
4. Нажми **«Загрузить распакованное расширение»** → выбери папку
5. Иконка 🍅 появится в панели расширений

## Использование

- **Кнопка «старт»** — запускает рабочий цикл (по умолчанию 25 минут)
- По завершении цикла появится уведомление и звук, запустится перерыв
- В иконке расширения отображается обратный отсчёт (красный = работа, зелёный = перерыв)
- Вкладка **«История»** — тепловая карта и статистика по дням недели

## Цветовая шкала (тепловая карта)

| Цвет | Помидоров в день |
|------|-----------------|
| ⬛ тёмный | 0 |
| 🟩 светло-зелёный | 1 |
| 🟩 зелёный | 2–3 |
| 🟩 насыщенный | 4–5 |
| 🟩 яркий | 6–8 |
| 🟩 максимум | 9+ |

Доступны 4 схемы: Green, Orange, Red, Ocean — с опцией инверсии.

## Экспорт и импорт

Во вкладке **Настройки** можно:
- Экспортировать всю историю в JSON-файл (резервная копия)
- Импортировать историю из JSON-файла

## Приватность

Все данные хранятся локально в браузере. Расширение не делает сетевых запросов и не отправляет никакие данные на внешние серверы.
