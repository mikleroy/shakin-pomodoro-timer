Privacy Policy

Shakin Pomodoro Timer — Chrome Extension — Last updated: April 2026

This extension does not collect, transmit, or share any personal data. Everything stays on your device.

Data Storage

The extension stores the following data locally in your browser using chrome.storage:

  • Pomodoro session history (date and count per day)
  • Timer settings (work and break duration)
  • UI preferences (theme, color scheme, language)
  • Current timer state (active session only)

This data never leaves your device. It is not sent to any server or third party.

No External Communication

The extension makes no network requests. There are no analytics, no tracking pixels, no crash reports, and no external APIs involved.

No Personal Information

The extension does not ask for, access, or store any personally identifiable information — no name, email, account, or browsing data.

Data Export

You can export your history as a JSON file at any time from the Settings tab. You can also delete all stored data from the same tab.

Changes to This Policy

If this policy changes in future versions, the updated date above will reflect that. Significant changes will be noted in the extension's update changelog.
